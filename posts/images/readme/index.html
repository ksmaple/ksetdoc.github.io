快速创建步骤

1. 访问 Excalidraw

打开 excalidraw.com

2. 使用库元素（可选）

点击左下角的「库」图标，可以添加：

  • 软件架构图标
  • 编程语言 Logo
  • 数据库图标

3. 配色方案建议

用途 颜色代码 说明
接口/抽象类 #a8d8ea 浅蓝色背景
具体类 #aa96da 浅紫色背景
重要元素 #fcbad3 浅粉色高亮
箭头/线条 #2c3e50 深灰色
文字 #2c3e50 深灰色

4. 字体设置

  • 标题：Comic Sans 或 Virgil，24-28px
  • 正文：Comic Sans 或 Virgil，18-20px
  • 小字说明：16px

5. 导出设置

  1. 点击右上角「导出」
  2. 选择 SVG 格式
  3. 勾选「包含背景」（如需白色背景）
  4. 导出后重命名为对应的文件名

6. 放置文件

将导出的 SVG 文件放入 content/posts/images/ 目录

流程图

使用矩形表示步骤，菱形表示判断，箭头表示流向。

对比图

左右或上下布局，使用不同颜色区分两个对比对象。

注意事项

  1. 保持简洁：一张图说明一个概念
  2. 统一风格：所有图表使用相同配色
  3. 适当留白：不要让图表过于拥挤
  4. 文字清晰：确保文字大小合适，易于阅读
  5. 文件名规范：使用小写字母，单词间用连字符分隔

推荐库

在 Excalidraw 中可以导入以下库：

  • Software Architecture: 系统架构图标
  • Database: 数据库相关图标
  • Programming Languages: 编程语言 Logo

导入方法：点击左下角「库」→「浏览库」→ 搜索并添加

快速开始

如果不想从头创建，可以：

  1. 在 Excalidraw 中搜索模板
  2. 或使用 AI 生成工具创建图表
  3. 截图后用 Excalidraw 的「从图片导入」功能
